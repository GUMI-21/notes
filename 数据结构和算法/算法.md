#### 分治法和递归
分而治之，分治法一般使用递归来求解
+ 递归
 斐波那契数
 ```go
 package algorithm  
  
func F(n int, a1, a2 int) int {  
   if n == 0 {  
      return a1  
   }  
   return F(n-1, a2, a1+a2)  
}
```

+ 动态规划

#### 算法复杂度和渐进符号
时间和空间，计算的速度越快 时间复杂度就越低。计算资源越少，空间复杂度就越低
循环N-1次，复杂度O(n)，计算1次O(1)
算法的优先级排列如下，一般排在上面的要优于排在下面的：
1. 常数复杂度：`O(1)`
2. 对数复杂度：`O(logn)`
3. 一次方复杂度：`O(n)`
4. 一次方乘对数复杂度：`O(nlogn)`
5. 乘方复杂度：`O(n^2)`，`O(n^3)`
6. 指数复杂度：`O(2^n)`
7. 阶乘复杂度：`O(n!)`
8. 无限大指数复杂度：`O(n^n)`
