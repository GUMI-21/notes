#### 2.1 变量名
+ 知识点
	+ 在 传统的 C 语言用法中，变量名使用小写字母，符号常量名全部使用大写字母
#### 2.2 数据类型及长度
+ C语言只提供了下列几种基本数据类型
	+ char 字符型 1个字节
	+ int 整型
	+ float 单精度浮点型
	+ double 双精度浮点型
#### 2.3常量
+ 以f或F结尾表示Float类型，l或L表示long double类型，u或U结尾表示无符号常量
+ 带前缀0的整数常量表示它为八进制形式，0x或0X则表示为十六进制
	+ 037 = 31 = 0x1f 
+ \ooo八进制 \xhh十六进制
